Privacy Policy – Transcendence

Last updated: 18/04/2026

1. Introduction

Transcendence is a web application developed as part of the 42 school curriculum. This Privacy Policy explains how we collect, use, and protect your personal data when you use the application.

2. Data We Collect

Account Information

  • Username
  • Email address (via OAuth provider)
  • Profile information (avatar, preferences)

Authentication Data

  • OAuth tokens (via 42 API)
  • Two-Factor Authentication data (if enabled)

Usage Data

  • Game history (Pong and Chess matches)
  • Scores and experience points
  • Connection logs (IP address, timestamps)

Communication Data

  • Messages sent through the chat system

3. How We Use Your Data

  • Provide and maintain the service
  • Enable real-time multiplayer features
  • Manage authentication and security
  • Store game history and statistics
  • Improve user experience

4. Data Sharing

We do not sell or share your personal data with third parties.

Data may be shared only with:

  • Authentication providers (42 API)
  • Infrastructure services strictly required for the application

5. Data Storage and Security

We implement security measures including:

  • Secure reverse proxy (Nginx)
  • Web Application Firewall (ModSecurity)
  • Secret management (HashiCorp Vault)
  • Containerized infrastructure (Docker)

Despite these measures, no system is completely secure.

6. Data Retention

Data is kept only for the duration necessary to operate the application.

Users may request deletion of their data.

7. Your Rights

Under applicable regulations (including GDPR), you have the right to:

  • Access your personal data
  • Request correction or deletion
  • Object to data processing

To exercise these rights, contact the project maintainers.

8. Third-Party Services

This application relies on:

  • 42 API (OAuth authentication)

We are not responsible for third-party privacy practices.

9. Changes to This Policy

This Privacy Policy may be updated at any time.

Users will be informed of significant changes.

10. Contact

For any questions regarding this Privacy Policy, please contact the project team.
